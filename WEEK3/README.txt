TAutoCorr.R:
It loads and examine and plot KeyWestAnnualMeanTemperature.Rdata, using load(). 
This is the temperature in Key West, Florida for the 20th century.
And it compute the appropriate correlation coefficient between successive years and store it
and repeat this calculation 10000 times.
